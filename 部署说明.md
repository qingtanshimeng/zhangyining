# 网站上线说明（5 分钟版）

## 1) 把二维码图片放到项目根目录

请把以下两张图复制到和 `index.html` 同级目录，并确保文件名一致：

- `douyin-qr.png`（抖音二维码，图一）
- `wechat-qr.png`（微信二维码，图二）

当前页面已经写好这两个相对路径，文件名正确就会自动显示。

## 2) 上线到静态托管平台（推荐）

可选平台：Vercel / Netlify / GitHub Pages。

最省事方式：

1. 新建一个 GitHub 仓库并上传当前目录所有文件
2. 在 Vercel 或 Netlify 导入该仓库
3. 框架选择 `Other` / `Static`
4. Build Command 留空，Output Directory 填 `.`
5. 点击 Deploy

## 3) 上线后要改的 2 个文件

请把你的真实域名替换以下占位地址：

- `robots.txt` 中的 `https://your-domain.com/sitemap.xml`
- `sitemap.xml` 中的 `https://your-domain.com/`

## 4) 上线后自检清单

- 首页是否能打开
- 抖音/小红书/B站链接是否可跳转
- 邮箱按钮是否能唤起发信
- 手机端显示是否正常
- 微信/抖音二维码是否加载成功
